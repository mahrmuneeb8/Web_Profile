Java Programming

Credit Hours: 3

This course introduces Java programming and object-oriented software development. Students learn how to build robust, platform-independent applications using Java.

Topics Covered

  • Introduction to Java
  • Variables and Data Types
  • Operators and Expressions
  • Input and Output
  • Conditional Statements
  • Loops
  • Methods
  • Arrays and Strings
  • Classes and Objects
  • Inheritance
  • Polymorphism
  • Exception Handling
  • File Handling
  • JSON Processing
  • JSX Basics
