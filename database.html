Database Development

Credit Hours: 3

This course introduces database concepts, design techniques, and database management systems. Students learn how to create, manage, and query databases using SQL and explore modern NoSQL database technologies.

Topics Covered

  • Introduction to Databases
  • Database Design
  • Entity Relationship (ER) Diagrams
  • Relational Database Model
  • SQL Queries
  • CRUD Operations
  • Joins and Subqueries
  • Normalization
  • Stored Procedures
  • Database Security
  • NoSQL Databases
  • Database Management Systems (DBMS)
